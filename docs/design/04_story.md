# 剧情设计文档

## 核心主题

**执念与放下**

修仙者追求长生，但长生的代价是什么？
你得到了力量，却失去了什么？
当一切尘埃落定，你是否还记得自己为何踏上这条路？

---

## 故事结构

### 全局脉络

```
【序章】落难
     ↓
【第一幕】归属 —— 找到家
     ↓
【第二幕】成长 —— 变得强
     ↓
【第三幕】裂痕 —— 发现真相
     ↓
【第四幕】抉择 —— 做出选择
     ↓
【第五幕】终局 —— 承受后果
     ↓
（轮回 / 结束）
```

---

## 序章：落难

### 开场

**时间**：游戏开始
**地点**：青云门外门，演武场

```
【开场白】

你叫什么名字？
> [玩家输入]

......

你站在演武场上，等待分配师父。

周围是和你一样的新弟子，有的紧张，有的期待。
你没什么感觉。能进青云门，已经是你这辈子最大的运气了。

三个月前，你还是街头的流浪儿。
如果不是那个路过的道士说你"有仙缘"，你可能一辈子都在要饭。

人群里，有人在冲你挥手。
是阿檀。
她跟你一起来的，虽然没有灵根修炼不了，但宗门收留了她做杂役。
她笑着做了个加油的手势。

你心里稍微安定了一些。
至少，还有她在。

"下一个——[玩家名字]！"
```

### 序章剧情点

```yaml
sequence:
  1. 分配师父:
    - 玩家被分配给云隐子
    - 云隐子没有出现，只派人带话
    - 其他弟子议论：云隐子是个怪人

  2. 初到云霞峰:
    - 认识大师兄苏云帆
    - 认识小师妹林雪瑶
    - 第一次见到师父云隐子

  3. 第一次考验:
    - 师父给了一个任务（采药/对练）
    - 玩家可能成功也可能失败
    - 无论如何，师父都只说一句"还行"

  4. 阿檀的处境:
    - 阿檀在外门做杂役
    - 有人欺负她
    - 玩家是否帮忙？（第一个选择）

选择后果:
  帮忙:
    - 可能挨打，可能惹麻烦
    - 阿檀感激，但也担心连累你
    - 某些人记住了你

  不帮:
    - 阿檀理解，但有些失落
    - 她开始学着自己解决问题
    - 你们之间有了第一道小小的裂痕
```

## 第一幕：归属

### 主题

**找到家的感觉。**

玩家从流浪儿变成青云门弟子，开始有了归属感。
这一幕是"温暖"的建立期，为后续的失去做铺垫。

### 时间跨度

游戏内约1-2年

### 关键剧情

```yaml
剧情线1【师门日常】:
  内容:
    - 跟着师父修炼
    - 和师兄师妹相处
    - 学习功法和武技
  情感:
    - 归属感
    - 被接纳的温暖
  关键场景:
    - 师父第一次夸你（虽然只是说"不错"）
    - 小师妹缠着你讲故事
    - 大师兄教你剑法

剧情线2【与阿檀的羁绊】:
  内容:
    - 定期去看阿檀
    - 她做了新衣服给你
    - 一起回忆流浪的日子
  情感:
    - 相依为命的情感深化
    - 开始有一点微妙的东西
  关键场景:
    - 中秋夜，你们坐在山坡上看月亮
    - 她说："以后我们会一直在一起吗？"

剧情线3【宗门事件】:
  内容:
    - 青云门内部的小事件
    - 可能是一次比武
    - 可能是一次任务
  作用:
    - 展现宗门的规矩和派系
    - 引入一些次要角色
    - 玩家开始理解这个世界的规则
```

### 第一幕结尾

**阿檀被发现有灵根。**

```
那天你去找阿檀，发现她不在。

一个外门弟子告诉你：
"你说的那个姑娘？被灵虚宗的人带走测试去了。"

你心里一沉。

灵虚宗？那是另一个宗门。
如果阿檀有灵根，她可以修炼了——这是好事。
但如果她去了灵虚宗...

你们就要分开了。

当晚，阿檀回来了。
她站在门口，表情复杂。

"他们说...我有灵根。"

你不知道该说什么。

"但是...我可以选择留下。"
她看着你，眼神里有期待，也有挣扎。
"留在青云门，做外门弟子。修炼会慢一些，但是..."

"但是什么？"

"但是可以和你在一起。"
```

**玩家选择：**
- 让她去灵虚宗（对她的未来好，但要分开）
- 让她留下（在一起，但她的修炼会受限）
- 让她自己决定（不干涉她的选择）

这个选择会影响阿檀的整条线。

## 第二幕：成长

### 主题

**变强，但代价是什么？**

玩家开始真正踏入修仙界，经历第一次生死危机，也发现这个世界的残酷。

### 时间跨度

游戏内约3-5年

### 关键剧情

```yaml
剧情线1【第一次历练】:
  内容:
    - 外出执行任务
    - 遇到真正的危险
    - 第一次面对生死
  情感:
    - 恐惧
    - 成长
    - 对实力的渴望
  关键事件:
    - 遇到一个必须杀的敌人
    - 第一次杀人的感受
    - 差点死掉，被人救（可能是师父，可能是他人）

剧情线2【阿檀的变化】:
  根据之前选择:
    A. 去了灵虚宗:
      - 很久没见面
      - 再见时她变得不一样了
      - 有些话不知道怎么说了
    B. 留在青云门:
      - 一起修炼，感情更深
      - 但她的修炼确实比较慢
      - 有人议论她是不是高攀
    C. 她自己选的:
      - 根据之前玩家的行为，她会做出选择
      - 玩家需要接受后果

剧情线3【身世之谜】:
  内容:
    - 偶然发现玉佩的秘密
    - 有人似乎认识这块玉佩
    - 开始追查自己的来历
  悬念:
    - 父母是谁？
    - 为什么被遗弃？
    - 那块玉佩代表什么？
```

### 第二幕高潮

**第一次与玄影相遇。**

玩家还不知道他是仇人。

```
那是一个普通的历练任务。

你在废弃的矿洞里发现了一个受伤的人。
他穿着黑衣，面容俊美，似笑非笑。

"小友，可否借一步说话？"

他没有恶意——至少表面上没有。
他说自己是散修，被仇家追杀，躲在这里。

你帮了他。

临别时，他看着你腰间的玉佩，眼神微变。
"这块玉佩...是你的？"

"是捡我的老人家给我的。怎么了？"

他笑了笑："没什么。你我有缘，他日再见。"

他走了。

你不知道的是——
他就是杀了你父母的人。
而他放过你，是因为时机未到。
```

## 第三幕：裂痕

### 主题

**真相开始浮现，平静被打破。**

### 时间跨度

游戏内约5-10年

### 关键剧情

```yaml
剧情线1【身世揭露】:
  内容:
    - 通过各种线索，拼凑出真相
    - 父母是谁（可能是某宗门的人，可能是散修，可能有更大的秘密）
    - 他们被谁杀了
  震撼:
    - 发现凶手就是那个黑衣人
    - 发现师父/掌门可能知道些什么
    - 世界不是你以为的那样

剧情线2【宗门变故】:
  内容:
    - 青云门出事了
    - 可能是魔道来袭
    - 可能是内部斗争
  影响:
    - 有人死了（可能是认识的人）
    - 有人叛变
    - 玩家必须做选择

剧情线3【关系危机】:
  阿檀:
    - 如果分开了：再见时两人都变了
    - 如果在一起：有人利用她来威胁玩家
  师父:
    - 玩家发现师父隐藏了什么
    - 对师父的信任动摇
  大师兄:
    - 可能开始表现出嫉妒
    - 在关键时刻会怎么选？
```

### 第三幕高潮

**师父被重伤/陨落。**

```
那一战，师父为了护住玩家，硬接了玄影一击。

金丹碎裂。

玩家看着倒在血泊中的师父，听他说：

"我年轻时...也有一个人为我这样做过..."
"你比我当年...强多了..."
"不要...走老路..."

他没说完。

玄影站在远处，微笑：
"云隐子，你护得了他一次，护得了一世吗？"
"小友，你想知道你父母是怎么死的吗？"

他走了。

留下玩家，面对师父的尸体（或重伤），和破碎的世界。
```

**玩家选择：**
- 追上去（可能送死）
- 留下来（救师父，但仇人逃走）
- 疯狂（失去理智，可能触发某些隐藏线）

## 第四幕：抉择

### 主题

**你要成为什么样的人？**

### 时间跨度

游戏内约10-30年（可能闭关突破）

### 核心问题

```yaml
正道与魔道:
  - 用正道的方法修炼，慢但稳
  - 用魔道的方法，快但有代价
  - 玄影的功法其实很强...

复仇与放下:
  - 追杀玄影（投入一切）
  - 先变强再说（可能失去机会）
  - 放下仇恨（可能吗？）

羁绊与力量:
  - 为了变强，要放弃什么？
  - 阿檀会等你多久？
  - 闭关十年出来，世界变成什么样？
```

### 关键剧情

```yaml
剧情线1【追寻真相】:
  内容:
    - 调查玄影的来历
    - 发现父母的秘密
    - 原来事情没那么简单
  可能的真相:
    - 父母可能不是完全无辜
    - 玄影可能有自己的理由
    - 有更大的阴谋在背后

剧情线2【最终修炼】:
  内容:
    - 玩家需要突破才能对抗玄影
    - 闭关修炼，代价是时间
    - 或者找到捷径，代价是其他
  选择:
    - 稳扎稳打（时间流逝）
    - 冒险突破（可能走火入魔）
    - 借助外力（欠下因果）

剧情线3【关系结算】:
  每条关系都要有结果:
    - 阿檀：道侣？分离？牺牲？
    - 师兄：和解？决裂？
    - 师妹：长大了，变成什么样？
    - 其他：每个重要NPC都有归宿
```

## 第五幕：终局

### 主题

**面对你的选择，承受你的结果。**

### 最终对决

```
玄影。

你们终于正面对上了。

这一战，可能在他的巢穴。
可能在某个秘境。
可能在青云门的废墟上。

他依然微笑：
"你终于来了。"
"你知道吗？你和你父亲真的很像。"
"他临死前的眼神，和你现在一模一样。"
```

**战斗结果可能：**
- 玩家赢，杀了他
- 玩家赢，放过他
- 玩家输，死亡→轮回
- 两败俱伤
- 第三方介入

### 结局分支

```yaml
结局1【复仇完成】:
  条件: 杀死玄影
  场景:
    - 仇人倒下
    - 但心里空了一块
    - 回头看，身边还有谁？
  变体:
    A. 身边有人（温暖的结局）
    B. 孑然一身（苍凉的结局）

结局2【原谅放下】:
  条件: 放过玄影+理解真相
  场景:
    - 放下执念
    - 但这条路真的对吗？
    - 玄影会怎么做？
  变体:
    A. 玄影洗心革面
    B. 玄影死于他人之手
    C. 玄影再次为恶

结局3【万劫不复】:
  条件: 走火入魔/堕入魔道
  场景:
    - 变成了自己曾经憎恨的人
    - 可能毁了一切
  结局:
    - 阿檀亲手杀了玩家
    - 或玩家杀了阿檀

结局4【轮回再起】:
  条件: 死亡
  场景:
    - 时间快进
    - 新的身份
    - 但因果未消
  开启: 第二周目

结局5【超脱】（隐藏）:
  条件: 多周目积累+特定选择
  场景:
    - 打破了某个更大的命运
    - 揭开了世界的真相
    - 真正的自由
```

## 隐藏剧情线

### 灵气衰竭

```yaml
线索:
  - 宗门秘档提到"灵气正在减少"
  - 高阶修士的焦虑
  - 某些区域已经无法修炼

真相:
  - 灵气衰竭是有人为之
  - 飞升者在抽取大陆灵气
  - 这个世界其实是一个"养殖场"

影响:
  - 揭开真相可能触发最终隐藏线
  - 与玄影的恩怨在这个真相面前变得渺小
```

### 师父的过去

```yaml
线索:
  - 师父为什么封印修为
  - 他口中"老朋友"是谁
  - 他为什么选择玩家

真相:
  - 师父年轻时的挚友为救他而死
  - 那个人和玩家有某种关联
  - 师父选择玩家，是在弥补遗憾

影响:
  - 理解师父之后，玩家对师门的感情更深
  - 继承师父遗志成为一种选择
```

### 轮回的秘密

```yaml
线索:
  - 某些NPC对玩家的态度异常
  - "似曾相识"的感觉
  - 梦中的碎片

真相:
  - 这不是玩家第一次轮回
  - 有人在操控轮回
  - 玩家每一世都在重复某种命运

影响:
  - 打破轮回成为最终目标
  - 需要多周目才能拼凑完整真相
```

## 剧情设计原则

```yaml
1. 不存在完美选择:
   每个选择都有代价
   让玩家纠结，而不是轻松选出"正确答案"

2. 情感优先于剧情:
   不是为了反转而反转
   每个转折都要有情感冲击力

3. 留白大于填满:
   不解释所有事情
   让玩家自己去想象、去猜测

4. 角色高于阴谋:
   复杂的阴谋不如复杂的人心
   玄影有血有肉，不是符号化的反派

5. 遗憾是好的结局:
   不是所有人都能救
   不是所有事都能圆满
   遗憾才让人记住
```
