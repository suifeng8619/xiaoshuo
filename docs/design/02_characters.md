# 角色设计文档

## 设计原则

每个重要角色都必须满足：

1. **有血有肉**：有欲望、有恐惧、有矛盾、有秘密
2. **会变化**：不是立牌坊，会因为经历而改变
3. **有立场**：在关键时刻会做出自己的选择，不一定符合玩家期望
4. **能失去**：可以死亡、可以离开、可以反目，这样玩家才会珍惜

## 核心角色

### 阿檀（沈檀儿）

**基础信息**
```yaml
name: 沈檀儿
nickname: 阿檀
age: 开局时16岁
gender: 女
origin: 孤儿，与玩家同为流浪儿出身
current_status: 青云门外门杂役弟子
```

**外貌**
```
清秀但不算绝色。常年流浪让她皮肤略显粗糙，但眼睛很亮。
习惯把头发扎成马尾，干净利落。
笑起来有两个浅浅的梨涡。
身量不高，看起来弱不禁风，实际上很能吃苦。
```

**性格层次**
```yaml
表面:
  - 温柔体贴，总是照顾别人
  - 乐观开朗，很少抱怨
  - 善解人意，懂得察言观色

内心:
  - 极度害怕被抛弃（童年阴影）
  - 自卑，觉得自己配不上玩家
  - 倔强，一旦决定的事不会改变

隐藏:
  - 有修炼天赋，但一直在压制
  - 害怕如果自己变强，和玩家的关系会改变
  - 对于"家"有执念，渴望稳定的归属
```

**核心羁绊**
```
与玩家：相依为命的情感。从流浪时期就在一起，玩家是她唯一的"家人"。
        表面是依赖，实际是互相依赖。
        她照顾玩家的起居，玩家给她安全感。
```

**人物弧光**
```
第一阶段【依附期】
  - 跟随玩家来到青云门
  - 甘愿做杂役，只要能陪在玩家身边
  - 全心全意支持玩家

第二阶段【觉醒期】
  - 被发现有灵根，可以修炼
  - 内心挣扎：修炼意味着和玩家走上同一条路，可能会改变两人关系
  - 开始有自己的际遇和选择

第三阶段【独立期】
  - 逐渐成长为独当一面的修士
  - 和玩家的关系可能：
    A. 成为道侣，并肩作战
    B. 道不同，渐行渐远
    C. 为保护玩家而牺牲
    D. 被仇人利用，成为威胁玩家的棋子
```

**关键记忆点**
```yaml
必须记住的事件:
  - 第一次相遇的情形
  - 玩家为她挨打的那天
  - 一起流浪的日子
  - 来到青云门的期待与不安
  - 玩家每一次重要选择时她的态度
  - 每一次分离和重逢

会影响态度的行为:
  - 玩家是否关心她的感受
  - 玩家是否在关键时刻保护她
  - 玩家是否对她坦诚
  - 玩家是否尊重她的选择
  - 玩家是否记得对她的承诺
```

**可能的结局**
```
【道侣】条件：长期关心、共同成长、关键时刻选择她
  - 并肩修仙，不离不弃
  - 可能共同面对天劫

【分离】条件：长期忽视、理念冲突、伤害过她
  - 和平分手或不欢而散
  - 可能在多年后重逢

【牺牲】条件：特定剧情触发 + 她对玩家感情深厚
  - 为救玩家而死
  - 成为玩家永远的遗憾

【黑化】条件：被仇人利用 + 玩家曾深深伤害过她
  - 被迫站在玩家对立面
  - 可能最终醒悟，也可能万劫不复
```

### 师父·云隐子

**基础信息**
```yaml
name: 云隐子
real_name: 秦远（极少人知道）
age: 外貌四十，实际三百余岁
gender: 男
status: 青云门长老，金丹期大圆满
position: 云霞峰峰主，玩家的授业恩师
```

**外貌**
```
面容清癯，两鬓微霜，目光深邃。
常年穿一身洗得发白的青袍，不修边幅。
看起来像个落魄的中年文士，毫无高人气派。
但出手时，眼神会变得锐利如剑。
```

**性格层次**
```yaml
表面:
  - 寡言少语，经常发呆
  - 对徒弟严格，少有夸赞
  - 不通人情世故，常被其他长老嘲笑

内心:
  - 极重情义，只是不善表达
  - 对自己要求极高，所以对徒弟也高要求
  - 内心柔软，见不得弱者受欺

隐藏:
  - 年轻时经历过刻骨铭心的往事
  - 选择玩家为徒，不只是因为天赋
  - 他的过去和玩家的身世可能有关联
```

**核心秘密**
```
云隐子三百年前曾是青云门最耀眼的天才，有望突破元婴。
但在一次任务中，他唯一的挚友为救他而死。
从此他性情大变，自封修为，不再争强好胜。
他在玩家身上看到了某个人的影子——那个人是谁？
```

**教导方式**
```yaml
理念: "我不会教你怎么变强，只会教你怎么活下去"

特点:
  - 从不手把手教，只给方向让玩家自己悟
  - 考验多于保护，相信玩家能扛过去
  - 关键时刻才会出现，但一定会出现
  - 很少夸奖，但每一句夸奖都有千钧之重

金句:
  - "修仙界没有公平可言，但你可以选择做一个公平的人"
  - "我教不了你什么，路是你自己走的"
  - "活着，比什么都重要"
  - "（第一次夸奖）不错，像我年轻时候一样莽撞"
```

**人物弧光**
```
第一阶段【严师】
  - 对玩家要求严格，冷面冷语
  - 玩家可能觉得他不近人情
  - 暗中保护玩家，但不让玩家知道

第二阶段【破冰】
  - 某次危机中，玩家窥见他的过去
  - 开始理解他的严厉是另一种关爱
  - 师徒关系真正建立

第三阶段【传承】
  - 在玩家达到某个节点时，传授真正的绝学
  - 可能会为保护玩家付出巨大代价
  - 可能会因为玩家的选择感到骄傲或失望

第四阶段【诀别或超越】
  - 师徒必有一别
  - 他死于护徒 / 他看着玩家超越自己 / 他的秘密揭开
```

**关键记忆点**
```yaml
必须记住:
  - 第一次见面玩家的表现
  - 玩家修炼中的每一次选择
  - 玩家是否遵守他的教导
  - 玩家如何对待他人
  - 玩家是否问过他的过去

会影响关系的行为:
  - 是否尊重师门规矩
  - 是否在困难面前放弃
  - 是否对同门手足保持善意
  - 是否追问他不愿提及的往事
```

### 仇人·玄影

**基础信息**
```yaml
name: 玄影（真名不详）
age: 外貌三十，实际不详
gender: 男
status: 幽冥教长老
cultivation: 元婴期初期
```

**外貌**
```
面容俊美，似笑非笑。
穿黑衣，声音低沉好听。
眼神深处有一丝不易察觉的疯狂。
初见让人觉得亲和，久处会感到彻骨的寒意。
```

**性格层次**
```yaml
表面:
  - 彬彬有礼，风度翩翩
  - 对敌人也保持微笑
  - 做事不紧不慢，胸有成竹

内心:
  - 极度执念于某个目标
  - 把所有人当作棋子
  - 对弱者有种病态的"怜悯"

隐藏:
  - 他的执念和玩家的身世有关
  - 他并非纯粹的恶人，有自己的"正义"
  - 他的过去和某个被掩埋的真相有关
```

**核心秘密**
```
玄影认识玩家的父母。
那块玉佩，他见过。
他追杀玩家父母，是因为他们手上有他想要的东西。
但这个"东西"究竟是什么？他想要做什么？
随着剧情推进，玩家会发现事情没那么简单。
```

**与玩家的羁绊**
```yaml
初期:
  - 玩家不知道他是仇人
  - 可能以"贵人"身份出现
  - 对玩家表现出异常的兴趣

中期:
  - 真相揭露，玩家得知他杀了自己的父母
  - 但动机成谜
  - 玩家开始追查真相

后期:
  - 真相复杂化：也许父母不是完全无辜的
  - 玩家面临选择：复仇？原谅？还是追求更大的真相？
```

**他不是纯粹的恶人**
```
这一点很重要。

他做过的恶事：
  - 杀了玩家的父母
  - 手段残忍，不择手段
  - 伤害过很多无辜的人

但是：
  - 他有自己的信念（虽然扭曲）
  - 他追求的目标可能不是"恶"
  - 他的过去可能有不得已的苦衷

玩家最终可以选择：
  - 杀了他（复仇完成，但真相可能永远埋没）
  - 放过他（他会成为什么？）
  - 与他合作（为了更大的目标？）
```

### 大师兄·苏云帆

**基础信息**
```yaml
name: 苏云帆
age: 外貌二十五，实际八十余岁
gender: 男
status: 云隐子座下大弟子，筑基期大圆满
position: 云霞峰大师兄
```

**外貌**
```
剑眉星目，身材修长。
典型的青年才俊长相，自带三分贵气。
穿戴整洁，举止从容。
笑起来让人如沐春风。
```

**性格层次**
```yaml
表面:
  - 温和有礼，关照师弟师妹
  - 能力出众，处事公正
  - 是师门的主心骨

内心:
  - 争强好胜，只是藏得好
  - 对自己要求极高，不容许失败
  - 渴望得到师父的认可

隐藏:
  - 多年卡在筑基大圆满，心生焦虑
  - 对玩家的天赋暗自嫉妒
  - 在巨大压力下可能做出什么选择？
```

**与玩家的关系**
```yaml
初期:
  - 照顾玩家，教导基础功法
  - 像大哥一样的存在
  - 玩家对他信任和尊敬

中期:
  - 玩家成长速度超过他
  - 他的态度开始微妙变化
  - 可能仍然支持玩家，也可能心生芥蒂

后期:
  走向A【成全】:
    - 他克服嫉妒，真心为玩家高兴
    - 成为玩家可靠的盟友
  走向B【沉沦】:
    - 被嫉妒蒙蔽，做出错误选择
    - 可能被利用，可能主动害人
  走向C【牺牲】:
    - 在关键时刻醒悟，以死谢罪
```

### 小师妹·林雪瑶

**基础信息**
```yaml
name: 林雪瑶
age: 开局时14岁
gender: 女
status: 云隐子座下小弟子
origin: 灵虚宗老朋友托付
```

**外貌**
```
灵动可爱，像个瓷娃娃。
眼睛很大，总是好奇地打量一切。
喜欢穿浅色衣服，走路蹦蹦跳跳。
```

**性格**
```yaml
表面:
  - 活泼开朗，叽叽喳喳
  - 黏人，特别喜欢跟着玩家
  - 有点小迷糊，经常闯祸

内心:
  - 比表面看起来聪明得多
  - 敏感，能察觉师门中的暗流
  - 害怕被抛弃，所以用活泼掩饰不安
```

**作用**
```
她是"温暖"的具象化。
在压抑的剧情中，她是调剂。
她可能长大后变得成熟，也可能...

注：她不是恋爱对象。她是"家人"的存在。
    如果她死了，会是玩家最痛的一刀。
```

## 角色记忆系统设计

每个重要NPC需要维护的数据：

```yaml
character_memory:
  # 基础信息
  id: unique_id
  name: 显示名

  # 与玩家的关系数值
  relationship:
    trust: 0-100        # 信任度
    affection: 0-100    # 好感度
    respect: 0-100      # 尊敬度
    fear: 0-100         # 畏惧度

  # 态度标签（可叠加）
  attitude_tags:
    - "视如亲人"
    - "暗生情愫"
    - "心存芥蒂"
    - "担忧其安危"

  # 记忆事件
  memories:
    - event_id: first_meeting
      description: "第一次见面的情形"
      emotional_impact: positive/negative/neutral
      importance: 1-10

  # 未解决的情感
  unresolved:
    - "玩家曾承诺带她离开，还没兑现"
    - "玩家受伤时她没能保护"

  # 秘密（玩家不知道的事）
  secrets:
    - revealed: false
      content: "她其实有灵根"
      trigger_condition: "特定剧情触发"

  # 当前目标
  current_goals:
    - "陪伴玩家"
    - "变得更强"
    - "找到归属感"
```
