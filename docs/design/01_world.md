# 世界观设计文档

## 核心理念

这个世界的核心不是"修仙"，而是"人"。

修仙只是背景，真正驱动故事的是：
- 人与人之间的羁绊
- 无法挽回的遗憾
- 必须做出的选择
- 时间带来的改变

---

## 天元大陆

### 基础设定

天元大陆，一片被灵气笼罩的广袤土地。

凡人在这片土地上繁衍生息，而修仙者则追寻长生大道。两个世界看似平行，却又紧密交织——每个修仙者都曾是凡人，每个凡人都可能踏入仙途。

### 地理格局

```
                    【北域·苦寒之地】
                         │
                    ┌────┴────┐
                    │ 万剑峰  │ ← 万剑宗
                    └────┬────┘
                         │
    【西域·荒漠】────┬────┼────┬────【东域·沃土】
                    │    │    │
               ┌────┴────┼────┴────┐
               │         │         │
          幽冥渊      天机城      青云山
          (魔道)     (散修)      (正道)
               │         │         │
               └────┬────┼────┬────┘
                    │    │    │
                    └────┴────┘
                         │
                    【南域·水乡】
```

**东域**：灵气最为充沛，正道宗门多聚于此。青云门、灵虚宗等大派林立。凡人王朝"大周"也在此地。

**西域**：荒漠戈壁，灵气稀薄但藏有上古遗迹。散修常来此寻找机缘，九死一生。

**北域**：苦寒之地，万剑宗独踞于此。剑修圣地，规矩森严，外人难入。

**南域**：水网密布，物产丰饶。小门派众多，也是凡人生活最安逸之处。

**中域**：天机城，大陆的十字路口。各方势力在此交汇，龙蛇混杂。

### 灵气与天地

灵气是修仙的根本。

灵气浓郁之地，修炼事半功倍，但也引来争夺。灵气稀薄之地，修士难以生存，却是凡人安居之所。

**灵脉**：大地之下的灵气通道。大宗门都建在灵脉之上。

**秘境**：天地自然形成的特殊空间，藏有机缘也藏有危险。有些秘境定期开启，有些则需要特殊方法才能进入。

**天劫**：修士突破大境界时引动的天地考验。渡过则脱胎换骨，失败则形神俱灭。

## 修仙体系

### 境界划分

```
凡人
  ↓
练气期（一至九层）
  ↓ ← 第一道坎：筑基丹或天赋异禀
筑基期（初期、中期、后期、大圆满）
  ↓ ← 第二道坎：需要机缘，九成修士止步于此
金丹期（初期、中期、后期、大圆满）
  ↓ ← 第三道坎：金丹九转，一步之差天壤之别
元婴期（初期、中期、后期、大圆满）
  ↓ ← 真正的强者，一方势力的顶梁柱
化神期
  ↓ ← 传说中的存在
渡劫期
  ↓ ← 天劫考验
大乘期
  ↓
飞升（传说）
```

**关键节点**：

- **练气到筑基**：需要"筑基丹"或极高天赋。这是第一道分水岭，九成修士卡在这里一辈子。
- **筑基到金丹**：需要悟道机缘。可能是一次生死战斗，可能是一段刻骨铭心的经历。
- **金丹到元婴**：元婴是"第二生命"，需要心境圆满，无心魔。
- **化神以上**：大陆上屈指可数，每一位都是活着的传说。

### 修炼道路

修仙不是只有一条路。

**剑修**：以剑入道，攻击凌厉。万剑宗是剑修圣地。
- 优势：战力强，同阶少有敌手
- 代价：心境要求高，执念太深易入魔

**丹修**：炼丹为主，辅助支援。灵药阁是丹修魁首。
- 优势：丹药无数，朋友遍天下
- 代价：战力较弱，需要同伴保护

**符修**：画符驱使，控制多变。
- 优势：手段多样，出其不意
- 代价：消耗极大，持久力差

**体修**：以身为炉，肉身成圣。
- 优势：防御惊人，越战越勇
- 代价：突破艰难，修炼痛苦

**魂修**：修炼神魂，精神攻击。
- 优势：神出鬼没，防不胜防
- 代价：被视为邪道，容易被围攻

**注**：道路不是选了就定死。可以兼修，但精力有限，样样通则样样松。

### 功法与武技

**功法**：修炼的根本，决定灵气运转方式。
- 功法有品阶：凡品、灵品、玄品、地品、天品
- 功法有属性：五行（金木水火土）、阴阳、无属性
- 功法可能冲突：修了A功法，就不能修B功法
- 功法可以传承：师父传给徒弟，是最大的信任

**武技/神通**：战斗的手段。
- 武技消耗灵力，威力取决于熟练度
- 神通是更高层次的手段，需要境界支撑
- 有些武技是绝学，只有特定门派才有

## 势力格局

### 正道三宗

**青云门**
- 位置：东域青云山
- 特点：历史悠久，底蕴深厚，注重心性修养
- 规矩：尊师重道，同门不可相残，不可滥杀无辜
- 掌门：清虚真人（化神期）
- 门派氛围：表面平和，内部派系林立
- 核心理念："道法自然，心正则仙途正"

**万剑宗**
- 位置：北域万剑峰
- 特点：剑修圣地，战力第一，规矩森严
- 规矩：剑不离身，战不退缩，宗门之命高于一切
- 宗主：剑尊（化神期巅峰）
- 门派氛围：冷肃，像军队
- 核心理念："剑之一道，唯快不破，唯心不移"

**灵虚宗**
- 位置：东域灵虚山
- 特点：女修居多，擅长幻术和治愈
- 规矩：不可入魔道，不可修邪功
- 掌门：灵虚仙子（化神期）
- 门派氛围：空灵出尘
- 核心理念："红尘炼心，虚极静笃"

### 魔道之首

**幽冥教**
- 位置：西域幽冥渊
- 特点：功法诡异，手段残忍，内部倾轧严重
- 规矩：实力为尊，没有规矩
- 教主：幽冥老祖（化神期，据说在闭关冲击渡劫）
- 门派氛围：人人自危，弱肉强食
- 核心理念："天道不仁，我命由我"

**注**：魔道不等于邪恶。只是修炼理念不同，更注重突破自我，不受世俗道德束缚。但这条路确实容易走偏，出恶人的概率更高。

### 其他势力

**散修**：没有门派的修士。自由但孤独，资源匮乏，往往需要冒险搏命。

**世家**：修仙家族，血脉传承。资源丰富但规矩多，家族利益高于个人。

**凡人王朝**：大周王朝统治凡人世界。修士一般不干涉凡人事务，但凡人权贵也会供养修士。

## 时间与轮回

### 修士寿命

```
练气期：150-200年
筑基期：300-500年
金丹期：800-1000年
元婴期：2000-3000年
化神期：5000年以上
渡劫期：万年以上
大乘期：不知其极
```

### 时间的残酷

这是游戏情感的重要来源。

- 你闭关十年突破筑基，出来后青梅竹马已经嫁人
- 你的凡人朋友一个个老去，你却还是少年模样
- 你的仇人寿命将尽，等你有能力报仇时，他已油尽灯枯
- 你苦苦追寻的人，早已飞升离去

时间让一切变得复杂。不是所有事都能等，不是所有人都能陪你走到最后。

### 轮回设定

天元大陆存在轮回。

普通人死后，魂归幽冥，转世投胎，前尘尽忘。

但修士不同。修士的执念太深，有些人即使转世也会保留一丝记忆——以"似曾相识"的形式。

这是玩家死亡后重新开始的设定基础：
- 你死了，但执念太深
- 你转世了，但某些人某些事，你总觉得在哪见过
- 某些选择，你会下意识做出——因为你上辈子就是这么选的
- 某些人对你态度奇怪——因为上辈子你们有过纠葛

轮回不是简单的"读档"，而是带着因果的重新开始。

## 世界的真相（隐藏设定）

以下内容玩家在游戏初期不会知道，但会随着剧情逐渐揭露：

1. **灵气在衰减**：天元大陆的灵气正在缓慢枯竭。千年前化神期遍地走，如今化神期屈指可数。大宗门对此讳莫如深。

2. **飞升的真相**：没有人知道飞升之后去了哪里。有人说是更高的仙界，有人说是虚无，有人说是陷阱。

3. **轮回的主宰**：轮回不是自然规律，而是有人在操控。这个"人"是谁？目的是什么？

4. **玩家的身世**：玩家不是普通的孤儿。玉佩的秘密将揭开一段被掩埋的历史。

这些真相会成为后期剧情的核心驱动力。

## 世界规则

### 铁律

1. **境界压制**：高两个大境界，可以碾压。高一个大境界，优势明显但可以战。同境界内，看天赋、功法、战斗经验。

2. **因果不虚**：杀人者终被杀。这不是诅咒，是这个世界的运行法则。修士修到高处，因果业力会成为心魔，阻碍突破。

3. **不可逆转**：死了就是死了，没有复活术。时间不能倒流，说过的话做过的事无法收回。

4. **天道无情**：没有谁是主角。实力不够强，就会死。选择错了，就要承担后果。

### 灰色地带

1. **正魔之分**：不是绝对的善恶。正道也有恶人，魔道也有真性情。

2. **恩怨纠缠**：很少有纯粹的坏人。你的仇人也许有自己的苦衷，你的恩人也许别有目的。

3. **两难抉择**：常常没有"正确答案"。救这个就要牺牲那个，成全别人就要委屈自己。

```
1枚上品灵石 = 100枚中品灵石 = 10000枚下品灵石
普通练气修士一月开销：约10枚下品灵石
一枚筑基丹：约500枚中品灵石（有价无市）
```

