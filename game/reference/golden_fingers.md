# 金手指详细说明

## 通用原则
- 穿越时绑定，无载体无意识
- 只有功能，没有人格
- 所有交易用灵石

---

## 1. 七日签到
- 每天签到一次，连续7天为周期
- 第7天必出稀有
- 断签重新计数
- 30天/100天/365天有成就奖励

### 核心原则：极致随机
- **万物皆可出**：世界上存在的任何物品都可能出现
- **不设固定池**：每次签到独立随机，不受之前影响
- **惊喜优先**：宁可出从没见过的东西，也不要重复
- **品质随机**：垃圾和宝物概率悬殊但都有可能
- **形式多样**：实物、情报、地图、符咒、生物、甚至一段记忆

### 随机性要求
1. 同一物品不应连续出现超过2次
2. 每10次签到至少出现3种新物品类型
3. 偶尔出现完全意想不到的东西（怪异物品、谜题、诅咒物等）
4. 奖励可以是残缺的、有副作用的、需要解谜的

### 品质概率（基础）
- 灰色（废物）：30% —— 完全无用或负面的东西
- 白色（垃圾/普通）：35%
- 绿色（有用）：20%
- 蓝色（稀有）：10%
- 紫色（珍贵）：4%
- 金色（传说）：0.9%
- 红色（神话）：0.1%

### 境界维度
每个物品除了品质，还有适用境界：
- 凡人级：凡人可用，对修士基本无效
- 炼气级：炼气期适用
- 筑基级：筑基期适用
- 结丹级：结丹期适用
- 元婴级及以上：极其罕见

**概率分布（按主角当前境界）：**
- 低于当前境界：10%（可能完全无用）
- 当前境界：75%
- 高于当前境界一阶：12%（可能暂时用不了）
- 高于当前境界两阶及以上：3%（大概率用不了，但可存着）

### 随机生成原则

**双重独立掷骰：**
1. 先掷品质骰（灰30%/白35%/绿20%/蓝10%/紫4%/金0.9%/红0.1%）
2. 再掷境界骰（低10%/当前75%/高一阶12%/高两阶+3%）
3. 两者独立组合，可能出现"金色凡人级"或"白色结丹级"

**物品类型不限定：**
- 不设固定物品池
- 世界观内任何东西都可能出现
- 实物、情报、地图、符咒、生物、记忆碎片等

**防重复规则：**
- 同一物品不连续出现超过2次
- 每10次签到至少出现3种不同物品类型
- 偶尔出现完全意想不到的东西

### 灰色物品示例
- 发霉的馒头
- 破烂的草鞋（单只）
- 一把泥土
- 死老鼠
- 写满字但看不懂的纸片
- 空的药瓶（有残留异味）
- 断裂的木簪
- 三天前的剩饭
- 一颗蛀牙
- 用过的符纸灰烬

### 第7天稀有日
- 品质至少绿色
- 有概率出现特殊奖励（组合物、成套物品、附带任务的物品等）

### 成就奖励
- 不预设具体奖励，根据当前剧情和角色状态随机生成
- 保证品质和意义与成就等级匹配
- 100天成就：紫色保底
- 365天成就：金色保底

## 2. 炼化抽奖
- 物品丢入炼化，按市场价1:1兑换灵石
- 消耗灵石抽奖，分档：1/10/100/1000/10000灵石...
- **灵石越多，抽到高境界和高品质物品概率越大**

### 各档位概率（已优化，好东西更容易出）

**1灵石档：**
- 品质：白40% / 绿35% / 蓝18% / 紫5% / 金1.8% / 红0.2%
- 境界：凡人级60% / 炼气级30% / 筑基级10%

**10灵石档：**
- 品质：白20% / 绿35% / 蓝28% / 紫13% / 金3.5% / 红0.5%
- 境界：凡人级30% / 炼气级55% / 筑基级15%

**100灵石档：**
- 品质：白5% / 绿20% / 蓝35% / 紫30% / 金9% / 红1%
- 境界：炼气级45% / 筑基级45% / 结丹级10%
- **每次必出至少1个蓝色以上物品**

**1000灵石档：**
- 品质：绿10% / 蓝25% / 紫40% / 金20% / 红5%
- 境界：筑基级45% / 结丹级45% / 元婴级10%
- **每次必出至少1个紫色以上物品**

**10000灵石档及以上：**
- 品质：蓝5% / 紫30% / 金45% / 红20%
- 境界：结丹级25% / 元婴级50% / 化神级25%
- **每次必出至少1个金色以上物品**

## 3. 修炼面板
- 可视化经验条
- 杀敌/修炼/任务获得经验
- 经验满自动突破，无失败风险
- 仍需时间积累，不能跳级

---

## 4. 简化炉
- 可简化：功法、法术、炼丹、炼器、阵法等**所有需要学习修炼的技艺**
- 效果：修炼/学习时间减少30-50%
- **威力不降低**，只是降低学习难度
- 简化后无法恢复原版

---

## 5. 合成台
- 同类物品3合1升级
- 3白→1绿→1蓝→1紫→1金
- 必须同类同属性
- 金色以上无法继续

---

## 6. 词条抽附
- 每天一次
- 从物品/生物抽取词条（抽取会销毁物品/需击杀生物）
- **可附加到任何物体或生物上**（包括自己）
- 同一目标最多3词条
- 品质：白/绿/蓝/紫/金/红

---

## 7. 系统回收商店
- 卖物品给系统：市场价×0.7
- 买系统商品：价格透明
- 商品每7天刷新
- 不会出现当前境界用不了的东西

---

## 8. 每日任务
- 每天3个任务，难度随机
- 类型：修炼/战斗/采集/探索/社交
- **奖励各种各样，不只是灵石，什么都有**
  - 简单：5-20灵石 / 低级丹药 / 凡物材料 / 情报
  - 中等：30-80灵石 / 法器 / 功法残篇 / 稀有材料
  - 困难：100-300灵石 / 法宝 / 完整功法 / 神秘物品
