# 物价参考表

## 锚定基准
辟谷丹（7天）1瓶10颗 = 1灵石

## 灵石分级
| 等级 | 地位 |
|------|------|
| 下品 | 基础货币，练气/筑基/金丹日常用 |
| 中品 | 战略物资，大宗交易 |
| 上品 | 顶级资源，元婴以上 |
| 极品 | 有价无市 |

**高品质灵石不换低品质**
