## 灵根分类
| 类型 | 稀有度 | 上限 |
|------|--------|------|
| 天灵根（单属性） | 万中无一 | 化神有望 |
| 异灵根（变异） | 罕见 | 不确定 |
| 双灵根 | 较少 | 元婴有望 |
| 三灵根 | 普通 | 金丹可期 |
| 四灵根 | 较差 | 筑基困难 |
| 五灵根 | 最差 | 炼气顶峰 |

## 修炼时间参考
凡人→炼气一层：感应灵气3天-1月，引气入体3-7天

炼气期小境界突破：
- 无资源：3-6个月
- 有月俸：2-3个月
- 有丹药：1-2个月
- 顶级条件：最快20-30天

筑基突破：
- 需筑基丹+闭关数月
- 成功率：天灵根80%，双灵根40-60%，三灵根20-30%
